Multi-cursor: Edita varias líneas a la vez

Mantén presionada la tecla Alt (Option en Mac) y haz clic en diferentes lugares para crear múltiples cursores. Puedes escribir en todos ellos a la vez. ¡Muy útil para renombrar variables o editar varias líneas similares!

Integración con Git: Controla tus versiones

VS Code tiene una excelente integración con Git. Puedes ver los cambios en tus archivos, hacer commits, crear ramas y mucho más, todo desde la interfaz de VS Code.
